# 🛡️ Task 2 - Phishing Email Analysis  
**Cyber Security Internship - Week 2 Assignment**

## 📌 Objective  
The goal of this task was to analyze a sample phishing email, identify phishing indicators, and understand common tactics used in email-based cyberattacks. This exercise builds skills in threat detection and email header analysis.

---

## 📨 Phishing Email Summary  
A fake PayPal notification email was analyzed. The email attempted to deceive the user into clicking a malicious link and downloading a harmful `.html` attachment.

---

## 🔍 Key Phishing Indicators Identified

- **Spoofed Sender Email**: `support@paypai-secure.com` (note the misspelled “PayPaI”)
- **Suspicious Link**: `http://paypai-secure-login.com/verify-now` (not an official PayPal domain)
- **Urgency in Message**: "Respond within 24 hours..."
- **Generic Greeting**: "Dear Customer" instead of the recipient's name
- **Unusual Attachment**: `Secure_Your_Account.html` file included
- **Grammar and tone inconsistencies**
  
---
